# Fear-Greed-Index-Predictor
This project predicts the stock market’s Fear &amp; Greed Index using a hybrid ML model. It combines financial indicators like volatility, price change, volume, and interest rates with sentiment analysis of market news to detect investor emotions such as fear, neutral, or greed in real time using Streamlit.

## Structure
- `app.py` — Streamlit frontend
- `main_logic.py` — FGI calculation logic
- `model/` — (optional) saved model file
